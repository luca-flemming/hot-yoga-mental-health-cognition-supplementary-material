# hot-yoga-mental-health-cognition-supplementary-material

Supplementary material for the bachelor thesis exploring the effects of hot yoga on mental health and cognition. This repository contains the Python code used for data preparation, statistical analysis and figure creation.

The following libraries were used: NumPy (version 1.26.4) Pingouin (version 0.5.4) Pandas (version 2.2.1) Seaborn (version 0.13.2 Matplotlib (version 3.8.3)
